## 1. 面向对象基础概念

### 1.1 什么是面向对象？

面向对象编程将现实世界的事物抽象为**对象**，每个对象包含：
- **属性**：描述对象的状态（如人的姓名、年龄）
- **方法**：定义对象的行为（如人走路、说话）

### 1.2 核心概念

| 概念     | 说明                     | 现实例子                                       |
| -------- | ------------------------ | ---------------------------------------------- |
| **类**   | 对象的蓝图或模板         | 汽车设计图                                     |
| **对象** | 类的实例                 | 根据设计图造出的真实汽车                       |
| **属性** | 对象的特征               | 汽车的颜色、品牌                               |
| **方法** | 对象的行为               | 汽车加速、刹车                                 |
| **继承** | 子类获得父类的属性和方法 | SUV继承汽车的通用特性                          |
| **封装** | 隐藏内部实现细节         | 汽车方向盘和踏板，内部引擎工作原理被隐藏       |
| **多态** | 同一接口的不同实现       | 汽车、自行车、卡车都有"刹车"方法，实现方式不同 |

## 2. 类与对象

### 2.1 定义类

使用 `class` 关键字定义类：

```python
# 最简单的类
class Car:
    pass  # 占位符

# 实际使用的类
class Student:
    """学生类 - 文档字符串"""
    
    # 类属性（所有实例共享）
    school = "第一中学"
    
    # 构造方法（初始化方法）
    def __init__(self, name, age, grade):
        # 实例属性（每个实例独有）
        self.name = name
        self.age = age
        self.grade = grade
    
    # 实例方法
    def study(self, subject):
        return f"{self.name}正在学习{subject}"
    
    def introduce(self):
        return f"我是{self.name}，{self.age}岁，{self.grade}年级"
```

### 2.2 创建对象（实例化）

```python
# 创建学生对象
student1 = Student("小明", 15, 9)
student2 = Student("小红", 14, 8)

# 访问属性和方法
print(student1.name)          # 小明
print(student2.introduce())   # 我是小红，14岁，8年级
print(student1.study("数学")) # 小明正在学习数学

# 访问类属性
print(Student.school)          # 第一中学（通过类访问）
print(student1.school)         # 第一中学（通过实例访问）
```

### 2.3 `self` 参数详解

- `self` 代表实例本身
- 在方法定义中必须是第一个参数
- 调用时自动传递，不需要手动传入

```python
class Person:
    def __init__(self, name):
        self.name = name  # self.name 是实例属性
    
    def say_hello(self):
        print(f"你好，我是{self.name}")
    
    def rename(self, new_name):
        self.name = new_name  # 修改实例属性

p = Person("张三")
p.say_hello()    # 你好，我是张三
p.rename("李四")
p.say_hello()    # 你好，我是李四
```

## 3. 属性与方法

### 3.1 实例属性 vs 类属性

```python
class Employee:
    # 类属性（所有员工共享）
    company = "科技公司"
    total_employees = 0
    
    def __init__(self, name, salary):
        # 实例属性（每个员工独有）
        self.name = name
        self.salary = salary
        # 修改类属性
        Employee.total_employees += 1
    
    def display_info(self):
        return f"{self.name} 在 {self.company} 工作，月薪 {self.salary}"

# 使用示例
emp1 = Employee("张三", 8000)
emp2 = Employee("李四", 9000)

print(emp1.display_info())  # 张三 在 科技公司 工作，月薪 8000
print(emp2.display_info())  # 李四 在 科技公司 工作，月薪 9000
print(f"员工总数: {Employee.total_employees}")  # 员工总数: 2

# 注意：通过实例修改类属性会创建实例属性
emp1.company = "其他公司"  # 创建实例属性，不影响类属性
print(emp1.company)        # 其他公司
print(emp2.company)        # 科技公司（类属性未变）
print(Employee.company)    # 科技公司（类属性未变）
```

### 3.2 实例方法、类方法、静态方法

```python
class MyClass:
    class_attr = "类属性"
    
    def __init__(self, value):
        self.value = value
    
    # 1. 实例方法（最常用）
    def instance_method(self):
        """可以访问实例属性和类属性"""
        return f"实例方法：{self.value}, {self.class_attr}"
    
    # 2. 类方法（使用 @classmethod 装饰器）
    @classmethod
    def class_method(cls):
        """只能访问类属性，不能访问实例属性"""
        return f"类方法：{cls.class_attr}"
    
    # 3. 静态方法（使用 @staticmethod 装饰器）
    @staticmethod
    def static_method(x, y):
        """独立函数，不访问类或实例的任何属性"""
        return f"静态方法：{x + y}"

# 使用示例
obj = MyClass("实例值")

print(obj.instance_method())  # 实例方法：实例值, 类属性
print(MyClass.class_method()) # 类方法：类属性
print(MyClass.static_method(3, 5))  # 静态方法：8

# 也可以从实例调用类方法和静态方法（不推荐）
print(obj.class_method())     # 类方法：类属性
print(obj.static_method(1,2)) # 静态方法：3
```

### 3.3 属性装饰器（@property）

实现属性的getter、setter和deleter，让方法像属性一样访问：

```python
class Temperature:
    def __init__(self, celsius):
        self._celsius = celsius
    
    # getter
    @property
    def celsius(self):
        """摄氏度"""
        return self._celsius
    
    # setter
    @celsius.setter
    def celsius(self, value):
        if value < -273.15:
            raise ValueError("温度不能低于绝对零度")
        self._celsius = value
    
    # 计算属性
    @property
    def fahrenheit(self):
        """华氏度（只读）"""
        return self._celsius * 9/5 + 32
    
    # deleter
    @celsius.deleter
    def celsius(self):
        print("删除温度属性")
        del self._celsius

# 使用示例
temp = Temperature(25)
print(temp.celsius)      # 25（像属性一样访问）
temp.celsius = 30        # 使用setter
print(temp.fahrenheit)   # 86.0

try:
    temp.celsius = -300  # 触发异常：温度不能低于绝对零度
except ValueError as e:
    print(f"错误：{e}")

# del temp.celsius  # 触发deleter
```

## 4. 继承

### 4.1 单继承

```python
# 父类（基类）
class Animal:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def eat(self):
        return f"{self.name}正在吃东西"
    
    def sleep(self):
        return f"{self.name}正在睡觉"
    
    def speak(self):
        return "动物发出声音"

# 子类（派生类）
class Dog(Animal):
    def __init__(self, name, age, breed):
        # 调用父类的构造方法
        super().__init__(name, age)
        self.breed = breed
    
    # 重写父类方法
    def speak(self):
        return f"{self.name}汪汪叫"
    
    # 子类特有的方法
    def fetch(self):
        return f"{self.name}正在玩接球游戏"

class Cat(Animal):
    def speak(self):
        return f"{self.name}喵喵叫"
    
    def climb(self):
        return f"{self.name}正在爬树"

# 使用示例
dog = Dog("旺财", 3, "金毛")
cat = Cat("咪咪", 2)

print(dog.eat())     # 旺财正在吃东西（继承）
print(dog.speak())   # 旺财汪汪叫（重写）
print(dog.fetch())   # 旺财正在玩接球游戏（特有）
print(cat.speak())   # 咪咪喵喵叫
print(cat.climb())   # 咪咪正在爬树
```

### 4.2 多重继承

Python支持多重继承（继承多个父类）：

```python
class Flyable:
    def fly(self):
        return "正在飞行"
    
    def move(self):
        return "飞行动物移动"

class Swimmable:
    def swim(self):
        return "正在游泳"
    
    def move(self):
        return "水生动物移动"

# 多重继承
class Duck(Flyable, Swimmable):
    def __init__(self, name):
        self.name = name
    
    # 可以重写方法
    def move(self):
        return f"{self.name}可以在水上游，也可以在天上飞"
    
    def quack(self):
        return "嘎嘎叫"

# 方法解析顺序（MRO - Method Resolution Order）
print(Duck.__mro__)  # 查看继承顺序

duck = Duck("唐老鸭")
print(duck.fly())    # 正在飞行（继承自Flyable）
print(duck.swim())   # 正在游泳（继承自Swimmable）
print(duck.move())   # 唐老鸭可以在水上游，也可以在天上飞（重写）
print(duck.quack())  # 嘎嘎叫（特有）

# 菱形继承问题：使用super()解决
class A:
    def method(self):
        print("A.method")
        super().method()

class B(A):
    def method(self):
        print("B.method")
        super().method()

class C(A):
    def method(self):
        print("C.method")
        super().method()

class D(B, C):
    def method(self):
        print("D.method")
        super().method()

d = D()
d.method()
# 输出顺序：D.method -> B.method -> C.method -> A.method
```

### 4.3 `super()` 函数详解

`super()` 用于调用父类方法，解决多重继承时的顺序问题：

```python
class Parent:
    def __init__(self, name):
        print(f"Parent __init__: {name}")
        self.name = name

class Child(Parent):
    def __init__(self, name, age):
        super().__init__(name)  # 调用父类构造方法
        print(f"Child __init__: {age}")
        self.age = age

class GrandChild(Child):
    def __init__(self, name, age, grade):
        super().__init__(name, age)  # 调用上一层父类
        print(f"GrandChild __init__: {grade}")
        self.grade = grade

gc = GrandChild("小明", 12, 6)
# 输出：
# Parent __init__: 小明
# Child __init__: 12
# GrandChild __init__: 6
```

## 5. 封装与访问控制

Python通过命名约定实现封装（没有严格的私有成员）：

### 5.1 访问控制级别

```python
class BankAccount:
    def __init__(self, account_number, balance):
        # 1. 公有属性（public）
        self.account_number = account_number
        
        # 2. 保护属性（protected）- 约定单下划线开头
        self._balance = balance
        
        # 3. 私有属性（private）- 名称修饰（name mangling）
        self.__secret_key = "123456"
    
    # 公有方法
    def deposit(self, amount):
        if amount > 0:
            self._balance += amount
            self.__log_transaction("存款", amount)
            return True
        return False
    
    def withdraw(self, amount):
        if 0 < amount <= self._balance:
            self._balance -= amount
            self.__log_transaction("取款", amount)
            return True
        return False
    
    # 保护方法（约定）
    def _calculate_interest(self):
        return self._balance * 0.02
    
    # 私有方法
    def __log_transaction(self, type, amount):
        print(f"[日志] {type}: {amount}, 余额: {self._balance}")
    
    # getter方法
    def get_balance(self):
        return self._balance

# 使用示例
account = BankAccount("12345678", 1000)

# 公有成员：可以直接访问
print(account.account_number)  # 12345678

# 保护成员：可以访问，但约定不要直接访问
print(account._balance)        # 1000（不推荐）
account._calculate_interest()  # 可以调用，但不推荐

# 私有成员：不能直接访问（名称修饰）
# print(account.__secret_key)   # AttributeError
# account.__log_transaction()   # AttributeError

# 但可以通过名称修饰访问（不推荐）
print(account._BankAccount__secret_key)  # 12345678

# 推荐通过公有方法访问
account.deposit(500)   # [日志] 存款: 500, 余额: 1500
account.withdraw(200)  # [日志] 取款: 200, 余额: 1300
print(account.get_balance())  # 1300
```

### 5.2 使用 `@property` 实现更好的封装

```python
class User:
    def __init__(self, username, email):
        self._username = username
        self._email = email
        self._password_hash = None
    
    @property
    def username(self):
        """用户名（只读）"""
        return self._username
    
    @property
    def email(self):
        return self._email
    
    @email.setter
    def email(self, value):
        """设置邮箱（带验证）"""
        if "@" not in value or "." not in value:
            raise ValueError("无效的邮箱地址")
        self._email = value
    
    @property
    def password(self):
        """密码不能读取"""
        raise AttributeError("密码不能直接读取")
    
    @password.setter
    def password(self, plain_password):
        """设置密码（自动加密）"""
        if len(plain_password) < 6:
            raise ValueError("密码长度至少6位")
        # 模拟加密过程
        self._password_hash = hash(plain_password)
        print("密码已加密存储")
    
    def verify_password(self, plain_password):
        """验证密码"""
        return hash(plain_password) == self._password_hash

# 使用示例
user = User("alice", "alice@example.com")
print(user.username)  # alice（只读）

user.email = "new_email@example.com"  # 使用setter
print(user.email)      # new_email@example.com

try:
    user.email = "invalid_email"  # 触发验证错误
except ValueError as e:
    print(f"错误：{e}")

# user.password  # AttributeError: 密码不能直接读取
user.password = "mypassword123"  # 密码已加密存储
print(user.verify_password("mypassword123"))  # True
```

## 6. 多态

多态允许不同类的对象对同一消息做出不同的响应：

### 6.1 方法重写实现多态

```python
class Shape:
    """形状基类"""
    def area(self):
        """计算面积 - 子类必须重写"""
        raise NotImplementedError("子类必须实现area方法")
    
    def perimeter(self):
        """计算周长 - 子类必须重写"""
        raise NotImplementedError("子类必须实现perimeter方法")

class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height
    
    def area(self):
        return self.width * self.height
    
    def perimeter(self):
        return 2 * (self.width + self.height)

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius
    
    def area(self):
        return 3.14159 * self.radius ** 2
    
    def perimeter(self):
        return 2 * 3.14159 * self.radius

class Triangle(Shape):
    def __init__(self, a, b, c):
        self.a, self.b, self.c = a, b, c
    
    def area(self):
        # 海伦公式
        s = (self.a + self.b + self.c) / 2
        return (s * (s - self.a) * (s - self.b) * (s - self.c)) ** 0.5
    
    def perimeter(self):
        return self.a + self.b + self.c

# 多态的使用：统一接口处理不同类型
def print_shape_info(shape):
    """多态函数：接受任何Shape子类"""
    print(f"形状: {shape.__class__.__name__}")
    print(f"面积: {shape.area():.2f}")
    print(f"周长: {shape.perimeter():.2f}")
    print("-" * 30)

# 创建不同的形状对象
shapes = [
    Rectangle(5, 3),
    Circle(4),
    Triangle(3, 4, 5)
]

# 统一处理
for shape in shapes:
    print_shape_info(shape)
# 输出：
# 形状: Rectangle
# 面积: 15.00
# 周长: 16.00
# ------------------------------
# 形状: Circle
# 面积: 50.27
# 周长: 25.13
# ------------------------------
# 形状: Triangle
# 面积: 6.00
# 周长: 12.00
# ------------------------------
```

### 6.2 鸭子类型（Duck Typing）

Python的多态更灵活：只要对象有相应的方法，就可以使用，不要求继承关系：

```python
class Duck:
    def quack(self):
        print("鸭子嘎嘎叫")
    def swim(self):
        print("鸭子游泳")

class Person:
    def quack(self):
        print("人学鸭子叫")
    def swim(self):
        print("人游泳")

def make_it_quack(obj):
    """不需要类型检查，只要对象有quack方法就可以"""
    obj.quack()
    obj.swim()

# 不同类的对象，只要实现了相同接口，都可以使用
duck = Duck()
person = Person()

make_it_quack(duck)    # 鸭子嘎嘎叫 / 鸭子游泳
make_it_quack(person)  # 人学鸭子叫 / 人游泳

# 更灵活的示例
class Robot:
    def quack(self):
        print("机器模拟鸭子叫")
    # 注意：机器人没有swim方法

# make_it_quack(Robot())  # 会报错：没有swim方法

def safe_make_it_quack(obj):
    """安全的鸭子类型使用"""
    if hasattr(obj, 'quack') and hasattr(obj, 'swim'):
        obj.quack()
        obj.swim()
    else:
        print("对象不符合鸭子类型要求")
```

## 7. 特殊方法（魔术方法）

特殊方法以双下划线开头和结尾，用于定制类的行为：

### 7.1 常用魔术方法

```python
class Vector:
    def __init__(self, x, y):
        """构造方法"""
        self.x = x
        self.y = y
    
    def __str__(self):
        """字符串表示（供用户阅读）"""
        return f"Vector({self.x}, {self.y})"
    
    def __repr__(self):
        """字符串表示（供开发者调试）"""
        return f"Vector(x={self.x}, y={self.y})"
    
    def __add__(self, other):
        """加法运算符 +"""
        if isinstance(other, Vector):
            return Vector(self.x + other.x, self.y + other.y)
        return NotImplemented
    
    def __sub__(self, other):
        """减法运算符 -"""
        if isinstance(other, Vector):
            return Vector(self.x - other.x, self.y - other.y)
        return NotImplemented
    
    def __mul__(self, scalar):
        """乘法运算符 *"""
        if isinstance(scalar, (int, float)):
            return Vector(self.x * scalar, self.y * scalar)
        return NotImplemented
    
    def __eq__(self, other):
        """相等运算符 =="""
        if isinstance(other, Vector):
            return self.x == other.x and self.y == other.y
        return False
    
    def __len__(self):
        """返回长度（用于len()函数）"""
        return int((self.x**2 + self.y**2) ** 0.5)
    
    def __getitem__(self, index):
        """索引访问"""
        if index == 0:
            return self.x
        elif index == 1:
            return self.y
        else:
            raise IndexError("索引超出范围")
    
    def __setitem__(self, index, value):
        """索引赋值"""
        if index == 0:
            self.x = value
        elif index == 1:
            self.y = value
        else:
            raise IndexError("索引超出范围")
    
    def __iter__(self):
        """使对象可迭代"""
        yield self.x
        yield self.y

# 使用示例
v1 = Vector(3, 4)
v2 = Vector(1, 2)

print(v1)                    # Vector(3, 4) - __str__
print(repr(v1))              # Vector(x=3, y=4) - __repr__
print(v1 + v2)               # Vector(4, 6) - __add__
print(v1 - v2)               # Vector(2, 2) - __sub__
print(v1 * 3)                # Vector(9, 12) - __mul__
print(v1 == Vector(3, 4))    # True - __eq__
print(len(v1))               # 5 - __len__
print(v1[0], v1[1])          # 3 4 - __getitem__

v1[0] = 10                   # __setitem__
print(v1)                    # Vector(10, 4)

for coord in v1:             # __iter__
    print(coord)             # 10 \n 4
```

### 7.2 上下文管理器（with语句）

```python
class FileManager:
    """实现上下文管理协议"""
    def __init__(self, filename, mode):
        self.filename = filename
        self.mode = mode
        self.file = None
    
    def __enter__(self):
        """进入with块时调用"""
        print(f"打开文件: {self.filename}")
        self.file = open(self.filename, self.mode)
        return self.file
    
    def __exit__(self, exc_type, exc_val, exc_tb):
        """退出with块时调用（无论是否发生异常）"""
        if self.file:
            print(f"关闭文件: {self.filename}")
            self.file.close()
        
        # 如果发生了异常，可以处理或返回True来抑制异常
        if exc_type:
            print(f"捕获到异常: {exc_type.__name__}: {exc_val}")
            return True  # 返回True表示异常已处理，不会向外抛出

    f.write("Hello, World!")
```

        self.isbn = isbn
        self.title = title
        self.author = author
        self.total_copies = total_copies
        self._available_copies = total_copies
        self._borrow_records = []  # 借阅记录
    
    @property
    def available_copies(self) -> int:
        """可用副本数（只读）"""
        return self._available_copies
    
    def borrow(self, user_id: str) -> bool:
        """借书"""
        if self._available_copies > 0:
            self._available_copies -= 1
            self._borrow_records.append({
                'user_id': user_id,
                'borrow_date': datetime.date.today(),
                'return_date': None
            })
            print(f"借书成功：{self.title}")
            return True
        else:
            print(f"抱歉，《{self.title}》已无可用副本")
            return False
    
    def return_book(self, user_id: str) -> bool:
        """还书"""
        # 查找未归还的记录
        for record in self._borrow_records:
            if record['user_id'] == user_id and record['return_date'] is None:
                record['return_date'] = datetime.date.today()
                self._available_copies += 1
                print(f"还书成功：{self.title}")
                return True
        
        print(f"未找到用户 {user_id} 借阅《{self.title}》的记录")
        return False
